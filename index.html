Introduction

HTML (Hypertext Markup Language) is a markup language used to create and structure content for the web. It uses various tags and attributes to define and organize text, images, multimedia, and other elements within a web page.

HTML tags come in various types and can aid in the design of your web page. Within HTML, there are four mandatory tags, namely html, title, head, and body, which need to be included in every web page.

< html >

< head >

< title >

< /title >

< /head >

< body >

< /body >

< html >

HTML

An HTML tag is a code element used to define the structure and content of a web page. Tags are enclosed in angle brackets and can have attributes that provide additional information about the content.

HTML tags help to define the purpose and meaning of different elements on a web page, such as headings, paragraphs, images, links, and multimedia. When a web browser reads an HTML document, it uses the tags to interpret and display the content correctly.

< !DOCTYPE html >

< html lang="en" >

Title

The title tag is an HTML element that defines the title of a web page. It is placed within the head section of an HTML document, and it does not display any visible content on the page. Instead, the title appears in the title bar or tab of a web browser when the page is loaded.

The title tag is important for search engine optimization as it provides a concise and descriptive summary of the content on the page, which can influence how search engines index and display the page in search results. Additionally, the title tag can help users to identify the relevance and purpose of a web page before they click on it.

< title > < /title >

Head

The head tag is an HTML element that contains metadata about a web page, which provides information about the page itself rather than its visible content. The head tag is placed before the body tag and contains elements such as the title tag, meta tags, links to stylesheets and scripts, and other data that helps web browsers and search engines to understand and display the page correctly.

The head tag does not display any visible content on the page, but it is important for search engine optimization and can improve the user experience by providing useful information about the page, such as its author, keywords, and description.

< head >

< meta="UTF-8" >

< link rel="stylesheet" href="styles.css" >

< script src="script.js" >< /script >

< /head >

Body

The body tag is an HTML element that contains the visible content of a web page, such as text, images, multimedia, and other elements that users can interact with. The body tag is placed after the head tag and is the main area where the content of the page is defined. HTML tags within the body tag define the structure of the content and specify how it should be displayed, such as headings, paragraphs, lists, tables, and forms.

The body tag can also contain links, buttons, and other interactive elements that allow users to navigate the page, submit data, or interact with other web applications. The content within the body tag is what users see and interact with when they visit a web page.

< body > < /body >

The body can have tags such as:

  • Headings Tag < h1 >< /h1 >
  • Paragraph Tag < p >< /p >
  • List Tags < ul >< /ul> < ol >< /ol> < li >< /li >
  • Anchor Tag < a >< /a >
  • Image Tag < img src="">
  • Table Tag < table >< /table >
  • Form Tag < form >< /form >
  • Div Tag < div >< /div >
  • Span Tag < span >< /span >
